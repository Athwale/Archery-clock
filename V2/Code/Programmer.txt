Chip: AVR, EEPROM: 128B, SRAM: 128B, flash: 2kB, THT ATTINY2313-20PU DIL20
Programmer: USBASP - USB 2.0 ISP programmer
Burner SW: AVRDUDE:
OS: Fedora

- Install:
sudo yum -y install avrdude avr-gcc avr-libc

- Compiling .c code:
This compiles and flashes a small LED blinking test program which blinks all LEDs connected to port B at 500ms intervals using the internal 8MHz rc oscillator.
avr-gcc -Wall -g -Os -mmcu=attiny2313 -o programmer_test.bin programmer_test.c

- Create .hex file 
avr-objcopy -j .text -j .data -O ihex programmer_test.bin test.hex

- Burn hex into microcontroller
avrdude -c usbasp -p attiny2313 -U flash:w:test.hex
